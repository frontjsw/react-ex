* History 분류 카테고리 코드 리스트
(분류 코드 마우스 hover 시 의미 출력)

Edit

  • ETT : 텍스트 수정 (Edit Text)
  • ECLA : 클래스 수정 (Edit Class)
  • ELAY : 구조 수정 (Edit Layout)
  • EIMG : 이미지 수정 (Edit Img)
  • EJS : 스크립트 수정 (Edit Script)
  • EBUG : 버그 수정 (Edit Bug)

Add

  • ATT : 텍스트 추가 (Add Text)
  • ACLA : 클래스 추가 (Add Class)
  • ALAY : 구조 추가 (Add Layout)
  • AIMG : 이미지 추가 (Add Img)
  • AJS : 스크립트 추가 (Add Script)
  • ATET : 테스트 코드 추가 (Add TestCode)

Delete

  • DTT : 텍스트 삭제 (Delete Text)
  • DCLA : 클래스 삭제 (Delete Class)
  • DLAY : 구조 삭제 (Delete Layout)
  • DIMG : 이미지 삭제 (Delete Img)
  • DJS : 스크립트 삭제 (Delete Script)
  • DTET : 테스트 코드 삭제 (Delete TestCode)

Renewal

  • RNEW : 페이지 전체 수정 (Renewal)
